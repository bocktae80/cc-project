# PDF 페이지 읽기 — pages 파라미터 실습 ⭐

> 500페이지짜리 책을 통째로 읽으라고 하면 힘들죠?
> "3장만 읽어줘"라고 하면 훨씬 쉬워요. PDF도 마찬가지!

클로드 코드의 Read 도구는 PDF 파일을 읽을 수 있습니다.
하지만 큰 PDF를 통째로 읽으면 실패할 수 있어요.
**pages 파라미터**로 원하는 페이지만 골라서 읽으면 됩니다!

---

## 리모컨 비유

```
TV 리모컨의 "페이지 이동" 버튼
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  📺 특정 채널로 이동  →  pages: "3"      (3페이지만 읽기)
  📺 구간 반복 재생    →  pages: "1-5"    (1~5페이지 읽기)
  📺 녹화 구간 선택    →  pages: "10-20"  (10~20페이지 읽기)
```

---

## pages 파라미터란?

클로드의 Read 도구에서 PDF를 읽을 때, **어떤 페이지를 읽을지** 지정하는 옵션입니다.

| 형식 | 의미 | 예시 |
|------|------|------|
| `"3"` | 3페이지만 | 특정 페이지 하나 |
| `"1-5"` | 1~5페이지 | 앞부분 읽기 |
| `"10-20"` | 10~20페이지 | 중간 부분 읽기 |

### 왜 필요할까?

| 상황 | pages 없이 | pages 있으면 |
|------|-----------|-------------|
| 10페이지 PDF | 전체 읽기 가능 | 필요 없음 |
| 50페이지 PDF | **실패!** (너무 큼) | `pages: "1-10"` 성공 |
| 200페이지 PDF | **실패!** | `pages: "45-60"` 성공 |

> **핵심 규칙:** 한 번에 **최대 20페이지**까지 읽을 수 있습니다.
> 10페이지가 넘는 PDF는 반드시 pages를 지정하세요!

## 따라하기

### 준비: sample PDF 파일

이 폴더에 아무 PDF 파일을 넣어주세요.

> **sample.pdf를 이 폴더에 넣어주세요.**
> 학교 교과서, 매뉴얼, 아무 PDF나 괜찮습니다.
> 없다면 인터넷에서 무료 PDF를 하나 다운로드하세요.

### 1단계: 클로드에게 PDF 읽기 요청

클로드 코드를 실행하고 이렇게 말해보세요:

```
sample.pdf의 첫 5페이지를 읽어줘
```

클로드는 내부적으로 Read 도구를 이렇게 사용합니다:

```
Read 도구 호출:
  file_path: "/경로/projects/10-cli-master/examples/pdf-reader/sample.pdf"
  pages: "1-5"
```

### 2단계: 특정 페이지만 읽기

```
sample.pdf의 3페이지만 읽어줘
```

```
Read 도구 호출:
  file_path: ".../sample.pdf"
  pages: "3"
```

### 3단계: 범위 지정해서 읽기

```
sample.pdf의 10페이지부터 15페이지까지 읽어줘
```

```
Read 도구 호출:
  file_path: ".../sample.pdf"
  pages: "10-15"
```

## 실전 대화 예시

### 예시 1: 목차 먼저 확인하기

```
You: report.pdf의 1페이지를 읽어서 목차를 보여줘

Claude: report.pdf의 1페이지를 읽어보겠습니다.

  [1페이지 내용]
  목차
  1장. 서론 .................. 3
  2장. 방법론 ................ 15
  3장. 결과 .................. 28
  4장. 결론 .................. 42

You: 3장 결과 부분을 읽어줘 (28-41페이지)

Claude: 28-41페이지를 읽어보겠습니다.
  [28-41페이지 내용 표시]
```

### 예시 2: 큰 PDF를 나눠서 읽기

```
You: manual.pdf가 총 몇 페이지인지 알 수 있어?

Claude: manual.pdf를 확인해보겠습니다.
  이 PDF는 총 85페이지입니다.

You: 처음 20페이지만 요약해줘

Claude: 1-20페이지를 읽어서 요약하겠습니다.
  [요약 내용]

You: 다음 20페이지도 요약해줘

Claude: 21-40페이지를 읽어서 요약하겠습니다.
  [요약 내용]
```

### 예시 3: 20페이지 초과 시

```
You: report.pdf의 1-30페이지를 읽어줘

Claude: 한 번에 최대 20페이지까지 읽을 수 있습니다.
  먼저 1-20페이지를 읽겠습니다.

  [1-20페이지 내용]

  나머지 21-30페이지도 읽을까요?
```

## pages 파라미터 규칙 정리

| 규칙 | 설명 |
|------|------|
| 최대 20페이지 | 한 번에 20페이지까지만 읽기 가능 |
| 10페이지 이하 PDF | pages 없이도 읽기 가능 |
| 10페이지 초과 PDF | **반드시** pages 지정 필요 |
| 형식 | `"3"`, `"1-5"`, `"10-20"` (문자열로 지정) |

---

## 자주 묻는 질문

| 질문 | 답변 |
|------|------|
| PDF 말고 다른 파일도 pages가 되나요? | 아니요, **PDF 전용** 파라미터입니다 |
| 한 번에 21페이지를 읽으면? | 실패합니다. 20페이지 이하로 나눠서 읽으세요 |
| 이미지가 많은 PDF도 되나요? | 텍스트 기반 PDF가 가장 잘 됩니다 |
| 암호 걸린 PDF는요? | 암호 해제가 필요합니다 |

  큰 PDF는 나눠서 읽자!
