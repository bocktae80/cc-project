# 에이전트 목록 & 대화 이름 변경 ⭐

> 리모컨으로 "어떤 채널이 있지?" 확인하듯, 클로드에게도 "어떤 에이전트가 있지?" 물어볼 수 있어요!

이 예제에서는 두 가지를 배웁니다:
1. `claude agents` — 사용 가능한 에이전트 목록 보기
2. `/rename` — 대화 이름 자동 생성하기

---

## 리모컨 비유

```
TV 리모컨의 "채널 목록" 버튼
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  📺 채널 목록 보기   →  claude agents    (어떤 에이전트가 있지?)
  📺 프로그램 이름표  →  /rename          (이 대화 이름을 바꿔줘!)
```

---

## 1부: claude agents — 에이전트 목록 보기

### 에이전트란?

클로드 코드에서 **에이전트**는 특정 역할을 수행하는 AI를 말해요.
일반 대화용, 코드 탐색용 등 **종류가 다릅니다**.

### 실행 방법

터미널에서 다음을 입력하세요:

```bash
claude agents
```

### 출력 예시

```
Available agents:

  Name             Description
  ────────────────────────────────────────────
  default          General-purpose coding agent
  code-review      Code review specialist
  planner          Planning and architecture agent
```

> **참고:** 표시되는 에이전트 목록은 설정과 프로젝트에 따라 다를 수 있습니다.

### 어떤 에이전트가 있나요?

| 에이전트 | 역할 | 비유 |
|---------|------|------|
| default | 기본 코딩 에이전트 | TV의 기본 채널 |
| (프로젝트별) | 프로젝트에 정의된 커스텀 에이전트 | 내가 추가한 즐겨찾기 채널 |

### 커스텀 에이전트는 어디서 정의하나요?

프로젝트의 `.claude/agents/` 폴더에 마크다운 파일로 만듭니다:

```
.claude/
└── agents/
    ├── reviewer.md      ← 코드 리뷰 에이전트
    └── planner.md       ← 기획 에이전트
```

> 커스텀 에이전트 만드는 법은 이 튜토리얼 범위를 넘어갑니다.
> 여기서는 **목록을 확인하는 방법**만 알면 됩니다!

## 2부: /rename — 대화 이름 자동 생성

### 문제 상황

클로드 코드로 대화를 하다 보면 대화 기록이 쌓이는데,
이름이 없으면 나중에 "이 대화에서 뭘 했더라?" 찾기 어려워요.

### /rename 이란?

대화 중에 `/rename`을 입력하면, 클로드가 **대화 내용을 분석**해서
적절한 이름을 자동으로 붙여줍니다.

### 사용 방법

클로드 코드 대화 중에 입력하세요:

```
/rename
```

### 실행 예시

```
You: 로그인 기능의 버그를 찾아줘

Claude: auth.ts 파일을 확인해보겠습니다...
[... 대화 진행 ...]

You: /rename

Claude: ✓ Conversation renamed to "로그인 버그 디버깅"
```

> `/rename`은 대화 **중간**에 사용합니다. 대화 내용이 있어야 이름을 생성할 수 있어요!

### 직접 이름 지정도 가능

자동 이름이 마음에 안 들면 직접 지정할 수도 있습니다:

```
/rename 내가 원하는 이름
```

```
You: /rename auth 모듈 리팩토링

Claude: ✓ Conversation renamed to "auth 모듈 리팩토링"
```

### 언제 쓰면 좋을까?

| 상황 | 추천 |
|------|------|
| 대화를 10분 이상 했을 때 | `/rename` (자동 생성) |
| 특정 주제의 대화일 때 | `/rename 주제 이름` (직접 지정) |
| 나중에 찾아볼 대화일 때 | 반드시 이름 붙이기! |

## 따라하기: 전체 실습

### 실습 1: 에이전트 확인

```bash
# 터미널에서 에이전트 목록 확인
claude agents
```

결과를 확인하고, 어떤 에이전트가 사용 가능한지 살펴보세요.

### 실습 2: 대화 후 이름 변경

```bash
# 1. 클로드 시작
claude

# 2. 아무 대화를 진행
You: 이 프로젝트의 구조를 설명해줘

# 3. 대화가 어느 정도 진행된 후
You: /rename

# 4. 자동 생성된 이름 확인!
```
