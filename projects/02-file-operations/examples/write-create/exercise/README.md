# Write 도구 연습 미션 🎯

> 클로드에게 부탁해서 파일을 직접 만들어보자!

## 미션 1: 자기소개 파일 만들기 ⭐

클로드에게 이렇게 말해보세요:

```
"introduce.txt 파일을 만들어줘.
내 이름은 ___이고, 취미는 ___이야.
좋아하는 과목은 ___이고, 장래 희망은 ___이야."
```

빈칸을 채워서 클로드에게 부탁하면, 클로드가 Write 도구로 파일을 만들어줍니다!

### ✅ 확인 방법
```
"introduce.txt 읽어줘"
```

---

## 미션 2: 구구단 파일 만들기 ⭐

클로드에게 이렇게 말해보세요:

```
"7단 구구단을 times-table.txt 파일로 만들어줘.
형식은 '7 x 1 = 7' 이런 식으로!"
```

### ✅ 확인 방법
```
"times-table.txt 읽어줘"
```

### 🌟 보너스 도전!
```
"2단부터 9단까지 전체 구구단을 full-times-table.txt로 만들어줘"
```

---

## 미션 3: 나만의 웹페이지 만들기 ⭐⭐

클로드에게 이렇게 말해보세요:

```
"my-page.html 파일을 만들어줘.
내용은:
- 제목: 나의 첫 웹페이지
- 배경색: 연한 파란색
- 가운데 정렬
- 이름, 취미, 좋아하는 것 3가지 리스트
- 이모지 포함해줘!"
```

### ✅ 확인 방법
1. `"my-page.html 읽어줘"` — 코드 확인
2. 브라우저에서 파일을 열어서 결과 확인!

### 🌟 보너스 도전!
```
"my-page.html에 어울리는 style.css도 같이 만들어줘"
```
