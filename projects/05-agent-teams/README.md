# Agent Teams — 팀으로 일하는 AI 에이전트 ⭐⭐⭐

> 혼자 공부할 때와 모둠 활동할 때, 어떤 차이가 있을까요?

혼자 공부하면 모든 걸 내가 해야 합니다. 수학 문제도 풀고, 자료도 찾고, 발표 준비도 혼자서요.
하지만 **모둠 활동**을 하면? 한 명은 자료 조사, 한 명은 정리, 한 명은 발표 준비를 맡아서 **동시에** 일할 수 있죠.

**Agent Teams**도 똑같습니다. 하나의 AI가 혼자 다 하는 대신, **여러 AI 에이전트가 팀을 이뤄서 동시에 작업**합니다.

```
🧑 혼자 공부                    👥 모둠 활동
──────────────                 ──────────────
나 → 자료조사                   팀장 → 작업 분배
나 → 정리                        ├── 조원A → 자료조사
나 → 발표준비                    ├── 조원B → 정리
(하나씩 순서대로...)              └── 조원C → 발표준비
                                (동시에 진행!)
```

---

## 이런 걸 배워요

- **Agent Teams이 뭔지** — 여러 AI가 팀으로 협력하는 구조
- **언제 쓰면 좋은지** — 혼자(서브에이전트) vs 팀(Agent Teams) 비교
- **직접 만들어보기** — 팀을 구성하고 Todo 웹앱을 함께 만드는 실습
- **도구 사용법** — TeamCreate, TaskCreate, SendMessage 등 핵심 도구

## 사전 준비

### 1. Claude Code 기본 사용법

이 튜토리얼은 **상급** 난이도입니다. 먼저 아래 내용을 알고 있어야 해요:

- Claude Code 설치 및 기본 사용법
- CLAUDE.md가 뭔지 ([메모리 시스템 튜토리얼](../01-memory-system/) 참고)
- 터미널(명령어) 기본 사용법

### 2. 환경변수 설정

Agent Teams는 아직 **실험 기능**이기 때문에, 직접 활성화해줘야 합니다.

터미널에서 아래 명령어를 실행하세요:

```bash
# 쉘 설정 파일 열기 (macOS 기준)
open ~/.zshrc

# 아래 줄을 추가하고 저장:
export CLAUDE_AGENT_TEAMS=1

# 터미널을 재시작하거나:
source ~/.zshrc
```

설정이 잘 됐는지 확인:

```bash
echo $CLAUDE_AGENT_TEAMS
# 1 이 나오면 성공!
```

> **주의사항**
>
> - **실험 기능입니다** — Agent Teams는 아직 정식 기능이 아닙니다. 동작이 바뀌거나 예상과 다를 수 있어요.
> - **토큰을 많이 사용합니다** — 에이전트가 여러 명이니까, 혼자 쓸 때보다 API 비용이 훨씬 많이 듭니다. 연습할 때는 작은 작업으로 시작하세요.
> - **글로벌 저장됩니다** — 팀/태스크 정보가 `~/.claude/` 폴더에 저장됩니다. 프로젝트 폴더가 아니라 내 컴퓨터 전체에서 공유되는 점을 기억하세요.
> - **정리를 꼭 하세요** — 사용이 끝나면 팀을 삭제해주세요. 안 그러면 다음에 Claude Code를 열 때 이전 팀이 남아있을 수 있습니다.
