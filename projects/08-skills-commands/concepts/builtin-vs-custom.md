# 기본 커맨드 vs 커스텀 스킬 🎮⌨️

> 게임에는 기본 키와 매크로가 있듯이, 클로드 코드에도 두 종류의 명령어가 있어요!

---

## 🎮 기본 커맨드 (Built-in Commands)

클로드 코드를 설치하면 **처음부터** 쓸 수 있는 명령어들이에요.
게임의 **기본 단축키**와 같죠!

### 주요 기본 커맨드

| 커맨드 | 설명 | 비유 |
|--------|------|------|
| `/help` | 사용 가능한 명령어 목록 | 📖 게임 설명서 |
| `/cost` | 현재 세션 비용 확인 | 💰 게임 코인 잔액 |
| `/compact` | 대화 내용 요약 (메모리 절약) | 🗜️ 인벤토리 정리 |
| `/clear` | 대화 초기화 | 🔄 게임 리셋 |
| `/model` | AI 모델 변경 | 🎭 캐릭터 변경 |
| `/init` | 프로젝트 초기 설정 | 🏗️ 새 게임 시작 |
| `/memory` | CLAUDE.md 메모리 편집 | 🧠 게임 설정 |
| `/bug` | 버그 리포트 | 🐛 고객센터 |
| `/review` | 코드 리뷰 요청 | 👀 감수 요청 |
| `/config` | 설정 변경 | ⚙️ 환경 설정 |

### 특징

- ✅ 설치하면 바로 사용 가능
- ❌ 내용을 수정할 수 없음
- ✅ 모든 프로젝트에서 동일하게 동작

---

## ⌨️ 커스텀 스킬 (Custom Skills)

**내가 직접 만드는** 명령어예요.
게임의 **매크로**처럼, 자주 하는 복잡한 동작을 한 번에 실행!

### 원리: 마크다운 파일 = 프롬프트 템플릿

커스텀 스킬의 정체는 그냥 **마크다운(.md) 파일**이에요!

```
.claude/commands/greet.md
```

이 파일 안에 클로드에게 시킬 작업을 적어두면,
`/greet`이라고 입력했을 때 클로드가 그 파일을 읽고 실행해요.

> 💡 **비유**: 레시피 카드를 적어두면, 요리사(클로드)가 그대로 따라하는 거예요!

### 특징

- ✅ 자유롭게 만들고 수정 가능
- ✅ `$ARGUMENTS`로 입력값을 받을 수 있음
- ✅ 팀원과 공유 가능 (git으로 관리)
- ✅ 프로젝트마다 다른 스킬 사용 가능

---

## 📂 스킬 파일 위치 (로딩 위치)

스킬 파일을 어디에 두느냐에 따라 **적용 범위**가 달라져요!

### 1. 프로젝트 스킬 (이 프로젝트에서만)

```
내-프로젝트/
└── .claude/
    └── commands/
        └── review.md    ← 이 프로젝트에서만 /review 사용 가능
```

> 🏠 **비유**: 우리 반 교실에만 있는 규칙

### 2. 전역 스킬 (모든 프로젝트에서)

```
~/.claude/
└── commands/
    └── review.md    ← 어떤 프로젝트에서든 /review 사용 가능
```

> 🌍 **비유**: 학교 전체에 적용되는 규칙

### 우선순위

프로젝트 스킬과 전역 스킬의 이름이 같으면? **프로젝트 스킬이 우선!**

```
프로젝트 .claude/commands/review.md  ← ✅ 이게 실행됨
전역     ~/.claude/commands/review.md ← ❌ 무시됨
```

> 💡 **비유**: 교실 규칙이 학교 규칙보다 우선하는 것처럼!

## 📊 한눈에 비교

```
┌─────────────────────────────────────────────────┐
│                 클로드 코드 명령어                   │
├────────────────────┬────────────────────────────┤
│   🎮 기본 커맨드     │     ⌨️ 커스텀 스킬           │
├────────────────────┼────────────────────────────┤
│ 내장되어 있음        │ 직접 만들어야 함              │
│ 수정 불가           │ 자유롭게 수정                 │
│ /help, /cost 등    │ /review, /explain 등        │
│ 고정된 동작          │ $ARGUMENTS로 유연            │
│ 모든 곳에서 동일     │ 프로젝트별로 다를 수 있음       │
└────────────────────┴────────────────────────────┘
```

---

## 💡 핵심 정리

1. **기본 커맨드** = 게임 기본 단축키 → 바로 사용 가능, 수정 불가
2. **커스텀 스킬** = 내가 만든 매크로 → 마크다운 파일로 자유롭게 제작
3. **스킬 파일 위치**: `.claude/commands/` (프로젝트) 또는 `~/.claude/commands/` (전역)
4. 같은 이름이면 **프로젝트 스킬이 우선**
5. 반복되는 지시는 스킬로 만들면 시간 절약! ⏰
