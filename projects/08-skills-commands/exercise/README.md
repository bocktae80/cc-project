## 🎯 미션 1: 번역 스킬 만들기 (⭐)

### 목표
한국어를 영어로 번역하는 `/translate` 스킬 만들기

### 요구사항
- `.claude/commands/translate.md` 파일 생성
- `$ARGUMENTS`로 번역할 한국어 문장 받기
- 출력 형식:
  - 🇰🇷 원문
  - 🇺🇸 번역
  - 📝 핵심 단어 3개 (발음 포함)

### 힌트

```markdown
다음 한국어를 영어로 번역해주세요: $ARGUMENTS

출력 형식:
- 🇰🇷 원문: (입력받은 한국어)
- 🇺🇸 번역: (영어 번역)
- 📝 핵심 단어:
  1. 단어 — 발음 — 뜻
  2. ...
```

### 테스트

```
/translate 오늘 날씨가 정말 좋아요
```

## 🎯 미션 2: TODO 수집 스킬 만들기 (⭐⭐)

### 목표
코드에서 TODO 주석을 찾아 정리하는 `/find-todos` 스킬 만들기

### 요구사항
- `.claude/commands/find-todos.md` 파일 생성
- `$ARGUMENTS`로 검색할 폴더/파일 경로 받기
- 출력 형식:
  - 📍 파일 경로와 줄 번호
  - 📝 TODO 내용
  - 🔢 우선순위 추정 (높음/중간/낮음)
  - 📊 전체 요약 (총 개수, 우선순위별 분포)

### 힌트

```markdown
다음 경로에서 TODO, FIXME, HACK 주석을 모두 찾아 정리해주세요: $ARGUMENTS

출력 형식:
## 📋 TODO 목록
- 📍 파일:줄번호
- 📝 내용
- 🔢 우선순위 (높음/중간/낮음)

## 📊 요약
- 총 N개 발견
- 높음: N개, 중간: N개, 낮음: N개
```

### 테스트

```
/find-todos src/
```

## 🎯 미션 3: 일일 보고서 스킬 만들기 (⭐⭐⭐)

### 목표
오늘의 git 변경사항을 보고서로 만드는 `/daily-report` 스킬 만들기

### 요구사항
- `.claude/commands/daily-report.md` 파일 생성
- `$ARGUMENTS`로 날짜 또는 추가 메모 받기
- 출력 형식:
  - 📅 날짜
  - 📝 오늘의 커밋 목록
  - 📁 변경된 파일 요약
  - 📊 통계 (커밋 수, 추가/삭제 줄 수)
  - 💬 한줄 코멘트

### 힌트

```markdown
오늘의 git 변경사항을 일일 보고서로 작성해주세요.

추가 정보: $ARGUMENTS

보고서 형식:
## 📅 일일 개발 보고서

### 📝 오늘의 커밋
(git log --oneline --since="today" 기반)

### 📁 변경 파일 요약
(어떤 파일들이 어떻게 바뀌었는지)

### 📊 통계
- 총 커밋: N개
- 변경 파일: N개
- 추가: +N줄 / 삭제: -N줄

### 💬 한줄 코멘트
(오늘의 작업에 대한 짧은 코멘트)
```

### 테스트

```
/daily-report 로그인 기능 개발 완료
```
