# 기본 커맨드 레퍼런스 📖

> 클로드 코드에 내장된 기본 커맨드 10가지를 한곳에 정리했어요!

---

## 📋 전체 목록

| # | 커맨드 | 설명 | 비유 |
|---|--------|------|------|
| 1 | `/help` | 도움말 보기 | 📖 게임 설명서 |
| 2 | `/cost` | 세션 비용 확인 | 💰 코인 잔액 |
| 3 | `/compact` | 대화 요약 | 🗜️ 인벤토리 정리 |
| 4 | `/clear` | 대화 초기화 | 🔄 게임 리셋 |
| 5 | `/model` | 모델 변경 | 🎭 캐릭터 선택 |
| 6 | `/init` | 프로젝트 초기 설정 | 🏗️ 새 게임 시작 |
| 7 | `/memory` | CLAUDE.md 편집 | 🧠 게임 설정 |
| 8 | `/bug` | 버그 리포트 | 🐛 고객센터 |
| 9 | `/review` | 코드 리뷰 | 👀 감수 요청 |
| 10 | `/config` | 설정 변경 | ⚙️ 환경 설정 |

## 📝 상세 설명

### 1. `/help`

```
/help
```

- **역할**: 사용 가능한 모든 커맨드와 스킬 목록을 보여줌
- **언제 쓰나**: 어떤 커맨드가 있는지 모를 때
- **팁**: 커스텀 스킬도 이 목록에 함께 나타나요!

---

### 2. `/cost`

```
/cost
```

- **역할**: 현재 세션의 토큰 사용량과 예상 비용 표시
- **언제 쓰나**: 비용이 궁금할 때, 대화가 길어졌을 때
- **표시 내용**: 입력 토큰, 출력 토큰, 총 비용

---

### 3. `/compact`

```
/compact
/compact 커스텀 요약 지시
```

- **역할**: 지금까지의 대화를 요약해서 컨텍스트 공간 절약
- **언제 쓰나**: 대화가 20개 이상, 주제가 바뀔 때
- **주의**: 요약은 남지만 원본 대화는 사라짐
- **팁**: 뒤에 추가 지시를 넣으면 요약 방식을 커스텀할 수 있어요

---

### 4. `/clear`

```
/clear
```

- **역할**: 대화 내용 완전 초기화
- **언제 쓰나**: 완전히 새로운 주제로 시작할 때
- **⚠️ 주의**: 돌이킬 수 없음! 모든 대화가 사라짐

---

### 5. `/model`

```
/model
/model claude-sonnet-4-5
```

- **역할**: AI 모델 변경
- **사용 가능한 모델**:
  - `claude-opus-4-6` — 가장 강력, 복잡한 작업용
  - `claude-sonnet-4-5` — 균형잡힌 선택 (기본)
  - `claude-haiku-4-5` — 빠르고 저렴, 간단한 작업용
- **팁**: 모델명을 바로 적으면 선택 화면 없이 변경돼요

---

### 6. `/init`

```
/init
```

- **역할**: 프로젝트에 `CLAUDE.md` 파일을 생성하여 초기 설정
- **언제 쓰나**: 새 프로젝트에서 클로드 코드를 처음 사용할 때
- **생성되는 것**: 프로젝트 설명, 규칙 등이 담긴 `CLAUDE.md`

---

### 7. `/memory`

```
/memory
```

- **역할**: `CLAUDE.md` 파일을 편집하여 클로드의 프로젝트 메모리 관리
- **언제 쓰나**: 클로드에게 기억시킬 규칙이나 맥락을 수정할 때
- **팁**: 프로젝트의 규칙, 컨벤션, 중요 정보를 여기에 적어두세요

---

### 8. `/bug`

```
/bug
```

- **역할**: 클로드 코드의 버그를 리포트
- **언제 쓰나**: 클로드 코드가 이상하게 동작할 때
- **결과**: GitHub 이슈로 연결됨

---

### 9. `/review`

```
/review
```

- **역할**: 현재 변경사항에 대한 코드 리뷰 요청
- **언제 쓰나**: git에 커밋하기 전 코드 검토가 필요할 때
- **팁**: 커스텀 `/review` 스킬을 만들면 이것을 덮어씀

---

### 10. `/config`

```
/config
```

- **역할**: 클로드 코드 설정 변경
- **설정 가능 항목**: 테마, 알림, 권한 등
- **언제 쓰나**: 동작 방식을 커스텀하고 싶을 때
