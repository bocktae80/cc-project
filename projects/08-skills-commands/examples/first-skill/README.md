---

## 📁 파일 구조

```
first-skill/
├── README.md                    ← 지금 보고 있는 파일
└── .claude/
    └── commands/
        └── greet.md             ← /greet 스킬 파일
```

---

## 🚀 사용법

### 1. 이 폴더에서 클로드 코드 실행

```bash
cd examples/first-skill
claude
```

### 2. 스킬 실행

```
/greet 철수
```

### 3. 다른 이름으로도 실행

```
/greet 영희
/greet 우리 반 친구들
/greet 세계
```

---

## 🔍 스킬 파일 살펴보기

`.claude/commands/greet.md` 파일을 열어보세요:

```markdown
사용자에게 친근하게 인사해주세요.

인사 대상: $ARGUMENTS

요구사항:
- 반말로 친근하게
- 이모지 포함
- 오늘의 격언 하나 추가
```

### 핵심 포인트

- `$ARGUMENTS` → `/greet` 뒤에 적은 내용이 여기에 들어가요
- 요구사항을 구체적으로 적을수록 결과가 일관적이에요
- 마크다운 문법을 자유롭게 사용할 수 있어요
