# 기본 커맨드 체험 예제 🎮

> 이 폴더에서 클로드 코드의 기본 커맨드를 직접 체험해보세요!

---

## 📋 체험 순서

### 1. 클로드 코드 실행

이 폴더에서 터미널을 열고:

```bash
claude
```

### 2. 기본 커맨드 체험

아래 커맨드를 **순서대로** 입력해보세요:

```
/help
```
→ 사용 가능한 커맨드 목록이 보이나요? ✅

```
/model
```
→ 사용 가능한 모델 목록이 보이나요? ✅

```
간단한 질문을 해보세요: "1+1은?"
```
→ 클로드가 답변했나요? ✅

```
/cost
```
→ 토큰 사용량과 비용이 보이나요? ✅

```
몇 가지 질문을 더 해보세요
```

```
/compact
```
→ 대화가 요약되었나요? ✅

```
/clear
```
→ 대화가 초기화되었나요? ✅

---

## 💡 팁

- `/help`는 언제든 쓸 수 있으니, 헷갈리면 먼저 쳐보세요
- `/cost`로 비용을 체크하는 습관을 들이면 좋아요
- `/compact`와 `/clear`의 차이를 확실히 기억하세요!
  - `/compact` = 요약해서 남김 📦
  - `/clear` = 완전히 삭제 🗑️
