# 04 — 웹 검색/페치 (~3분)

> Hook: 클로드가 인터넷 검색을 한다고?

---

## 1. Hook (30초)

[화면] 클로드 코드 터미널

[나레이션]
클로드가 학습한 정보에는 시간 제한이 있어요.
"React 최신 버전이 뭐야?"라고 물으면 오래된 답을 줄 수도 있죠.

[화면] "최신 정보를 모릅니다" 같은 답변 예시

[나레이션]
하지만! 클로드 코드에는 인터넷을 검색하는 도구가 있습니다.
마치 비서에게 "이거 좀 검색해와"라고 시키는 것처럼요!

---

## 2. Concept (1분)

[전환] 비서 비유 다이어그램

[화면] 비유 화면
```
🔍 WebSearch = "이 주제에 대해 검색해와!" (검색 비서)
📄 WebFetch  = "이 웹페이지 가서 내용 정리해와!" (방문 비서)
```

[나레이션]
**WebSearch**는 검색엔진에서 결과를 가져와요.
검색어를 주면 관련 링크들을 찾아줍니다.

**WebFetch**는 특정 URL에 가서 내용을 읽어와요.
웹페이지의 내용을 마크다운으로 정리해줍니다.

[강조] ⚠️ 주의사항 표시

[나레이션]
중요한 제한! 로그인이 필요한 사이트는 접근할 수 없어요.
Google Docs, Notion, Jira 같은 건 안 됩니다.

---

## 3. Demo (1분)

[화면] 터미널

[행동] `claude` 실행 → "클로드 코드 최신 업데이트 검색해줘" 입력

[화면] WebSearch 실행 → 검색 결과 목록 표시

[나레이션]
WebSearch가 최신 검색 결과를 가져왔어요!
여기서 관심 있는 링크를 골라서...

[행동] "이 페이지 내용 요약해줘" (URL 제시)

[화면] WebFetch 실행 → 페이지 내용 분석

[나레이션]
WebFetch가 페이지에 가서 내용을 읽고 요약해줬습니다!
이렇게 두 도구를 합치면 실시간 리서치가 가능해요.

---

## 4. Recap (30초)

[전환] 요약 화면

[나레이션]
오늘 배운 것!

1️⃣ **WebSearch** = 인터넷 검색 (검색어 → 결과 목록)
2️⃣ **WebFetch** = 웹페이지 분석 (URL → 내용 정리)
3️⃣ **조합** = 검색 → URL 발견 → 페이지 분석 = 실시간 리서치!

[화면] 다음 편 예고 — "05편: AI 하나로 부족하면 팀으로!"

[나레이션]
다음 편에서는 여러 AI 에이전트를 팀으로 만들어 협업하는 방법을 알아봅니다!
