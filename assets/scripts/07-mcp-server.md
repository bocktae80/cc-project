# 07 — MCP 서버 (~7분)

> Hook: 클로드가 슬랙, 깃허브를 직접 다룬다면?

---

## 1. Hook (30초)

[화면] 여러 앱 아이콘 (Slack, GitHub, 파일 관리자)

[나레이션]
클로드 코드에 내장된 도구는 훌륭하지만...
슬랙 메시지를 보내거나, 깃허브 이슈를 만들거나,
데이터베이스를 검색하는 건 못 해요.

[화면] 스마트폰 앱 스토어 비유 등장

[나레이션]
스마트폰에 앱을 설치하듯, 클로드에도 **새 기능을 추가**할 수 있다면요?
그게 바로 MCP 서버입니다!

---

## 2. Concept (2분)

[전환] USB 허브 비유

[화면] USB 허브 다이어그램
```
🖥️ 컴퓨터 (클로드 코드)
    │
    🔌 USB 허브 (MCP 프로토콜)
    ├── 🖨️ 프린터 (파일시스템 서버)
    ├── 📷 카메라 (GitHub 서버)
    ├── 🎮 게임패드 (Slack 서버)
    └── 🔧 나만의 장치 (커스텀 서버)
```

[나레이션]
MCP는 **Model Context Protocol**의 약자예요.
컴퓨터에 USB로 장치를 연결하듯,
클로드에 **MCP 서버**를 연결하면 새 도구가 생기는 거예요!

[전환] 식당 비유

[화면] 식당 비유 다이어그램
```
🍽️ 식당
├── 📋 메뉴판 = tools/list (사용 가능한 도구 목록)
├── 📝 주문서 = tools/call (도구 실행 요청)
├── 👨‍🍳 주방 = MCP 서버 (기능 제공)
└── 🧑 손님 = 클로드 (기능 사용)
```

[나레이션]
식당에 비유하면, MCP 서버는 **주방**이에요.
메뉴판(tools/list)을 보고 원하는 걸 주문(tools/call)하면,
주방에서 만들어서 가져다줍니다!

[강조] 통신 방식: stdin/stdout (파이프)

[나레이션]
클로드와 MCP 서버는 터미널의 입출력으로 대화해요.
마치 전화 통화처럼, 요청을 보내면 결과가 돌아오는 거죠.

## 3. Demo (3분 30초)

### 데모 1: 파일시스템 MCP 서버 (1분)

[화면] 터미널

[행동] `claude mcp add filesystem -- npx -y @anthropic/mcp-server-filesystem /path/to/folder`

[나레이션]
파일시스템 서버를 추가합니다. 이 서버는 특정 폴더의 파일을 관리해줘요.

[행동] `claude mcp list`

[화면] 등록된 MCP 서버 목록

[행동] `claude` 실행 → "sandbox 폴더에 어떤 파일이 있어?" 입력

[화면] MCP 서버가 폴더 내용을 보여줌

### 데모 2: 나만의 MCP 서버 만들기 (2분 30초)

[나레이션]
이번엔 직접 MCP 서버를 만들어봅시다!
현재 시간을 알려주는 서버예요.

[화면] my-time-server.js 코드

[강조] McpServer, server.tool(), StdioServerTransport

[나레이션]
코드는 간단해요. 서버를 만들고, 도구를 등록하고, 시작!

[행동] `npm install` → MCP 서버 등록

[화면] settings.json에 mcpServers 설정 추가

[행동] `claude` 실행 → "지금 몇 시야?" 입력

[화면] MCP 서버가 현재 시간 반환

[강조] 클로드가 MCP 서버의 도구를 사용하는 모습

[나레이션]
클로드가 우리가 만든 MCP 서버한테 시간을 물어봤어요!
이렇게 원하는 기능을 직접 만들어 연결할 수 있습니다.
