## 2. Concept (1분 30초)

[전환] 게임 단축키 비유

[화면] 게임 컨트롤러 + 키보드
```
🎮 게임 기본 키     ⌨️ 내가 만든 매크로
F1 = 도움말          Ctrl+1 = 필살기 콤보
F5 = 저장            Ctrl+2 = 자동 회복
F9 = 불러오기        Ctrl+3 = 아이템 정리
```

[나레이션]
게임에 기본 단축키가 있고, 매크로도 만들 수 있잖아요?
클로드도 마찬가지!

[화면] 비교표
```
기본 커맨드 (내장)          커스텀 스킬 (내가 만듦)
/help = 도움말              /review = 코드 리뷰
/cost = 비용 확인           /explain = 코드 설명
/compact = 대화 요약        /test-gen = 테스트 생성
```

[나레이션]
기본 커맨드는 클로드에 이미 내장되어 있어요.
커스텀 스킬은 `.claude/commands/` 폴더에 마크다운 파일을 만들면 끝!

[강조] `.claude/commands/이름.md` = 프롬프트 템플릿

[나레이션]
스킬 파일은 그냥 **마크다운**이에요.
클로드에게 보내는 지시서를 파일로 저장하는 거죠.
`$ARGUMENTS`라고 쓰면, 사용자가 뒤에 입력한 내용이 들어갑니다!

## 3. Demo (2분)

### 데모 1: 기본 커맨드 체험 (30초)

[화면] 터미널

[행동] `claude` 실행 → `/help` 입력

[화면] 도움말 표시

[행동] `/cost` 입력

[화면] 현재 세션 비용 표시

[나레이션]
기본 커맨드는 바로 사용할 수 있어요.

### 데모 2: 첫 번째 스킬 만들기 (1분 30초)

[나레이션]
이제 나만의 스킬을 만들어봅시다!
친구에게 인사하는 스킬이에요.

[화면] `.claude/commands/` 폴더 생성

[행동] greet.md 파일 작성

[화면] greet.md 내용
```markdown
사용자에게 친근하게 인사해주세요.
인사 대상: $ARGUMENTS
요구사항:
- 반말로 친근하게
- 이모지 포함
```

[행동] 클로드에서 `/greet 철수` 실행

[화면] 클로드가 "철수야 안녕! 😊..." 식으로 인사

[강조] $ARGUMENTS → "철수" 로 치환!

[나레이션]
`/greet` 뒤에 "철수"를 넣으면, `$ARGUMENTS`가 "철수"로 바뀌어서
클로드가 철수에게 인사해줍니다!

### 데모 3: 실용 스킬 (30초)

[행동] `/review app.js` 실행

[화면] 심각도 표시된 코드 리뷰 결과

[나레이션]
실전에서는 코드 리뷰, 테스트 생성 같은 스킬이 정말 유용해요!
