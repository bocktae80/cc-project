[나레이션]
만약 클로드가 이 규칙들을 **기억**할 수 있다면요?

---

## 2. Concept (2분)

[전환] 비유 다이어그램으로 전환

[나레이션]
클로드의 메모리 시스템은 학교에 비유할 수 있어요.

[화면] 비유 다이어그램
```
📋 교실 게시판 = CLAUDE.md (팀 전체가 봄)
🗄️ 내 서랍 메모 = CLAUDE.local.md (나만 봄)
📓 항상 갖고 다니는 수첩 = ~/.claude/CLAUDE.md (어디서든 적용)
🤖 클로드의 일기장 = 자동 메모리 (알아서 기록)
```

[나레이션]
가장 기본은 `CLAUDE.md`!
프로젝트 폴더에 이 파일을 만들면, 클로드가 **자동으로** 읽어서 규칙을 따릅니다.

[화면] CLAUDE.md 파일 내용 예시

[나레이션]
여러 곳에 설정이 있으면?
**가까운 것이 우선**이에요. 프로젝트 규칙이 전역 규칙보다 센 거죠.

[강조] 우선순위: 프로젝트 > 워크스페이스 > 유저

---

## 3. Demo (2분)

[화면] 터미널 — before/ 폴더

[행동] `cd before/` 이동

[나레이션]
CLAUDE.md가 없는 폴더에서 시작합니다.

[행동] `claude` 실행 → "인사 함수 만들어줘" 입력

[화면] 클로드가 영어 주석으로 답변

[나레이션]
보이시죠? 클로드는 아무 맥락이 없어서 영어로 답했어요.

[전환] after/ 폴더로 이동

[행동] `cd ../after/` 이동 → `claude` 실행

[강조] 터미널에 "CLAUDE.md loaded" 표시

[행동] 같은 질문: "인사 함수 만들어줘"

[화면] 클로드가 한국어 주석 + camelCase + JSDoc으로 답변

[나레이션]
같은 질문인데 CLAUDE.md 하나로 이렇게 달라집니다!
