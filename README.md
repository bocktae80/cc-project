# CC Project

클로드 코드(Claude Code)의 기능을 배우고 활용하는 학습 프로젝트입니다.

## 누구를 위한 프로젝트인가요?

- 코딩을 처음 접하는 중학생
- 개발 경험이 없는 팀원
- 클로드 코드를 처음 사용하는 사람

## 구조

| 폴더 | 설명 |
|------|------|
| `projects/` | 기능별 예제 프로젝트 (11개 + 번외 1개) |
| [`studio/`](studio/index.html) | 학습 대시보드 — 브라우저에서 `studio/index.html`을 열어보세요 |
| `assets/` | 스크린샷, 다이어그램, 영상 대본 |
| `workbook/` | 프로젝트 상태 관리 |
